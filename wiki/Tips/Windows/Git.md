# ssh 経由でリモートのリポジトリにアクセス。

ssh 経由でリモートのリポジトリにアクセスする際には、以下のように指定した。

    ssh://user@domain:port/path/to/repo

鍵は以下のディレクトリに置いた。

    C:\Users\username\.ssh
